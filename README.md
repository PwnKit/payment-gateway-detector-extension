# Payment Gateway Detector for Chrome

Detect payment gateways on checkout pages in one click.

This Chrome extension scans the current page and related same-origin frames to identify gateways such as Stripe, PayPal, Razorpay, Square, and others using stronger fingerprints, delayed rescans, and stricter matching rules.

## Features
- Detects common payment gateways
- Same-origin iframe scanning
- Delayed rescans for dynamic checkouts
- Popup-based quick results
- Lightweight and easy to install

# Payment Gateway Detector max accurate

This keeps the stable popup-scanning architecture and improves only accuracy:
- stronger gateway fingerprints
- stricter matching for generic providers
- extra checkout/form/token clues
- same-origin iframe scanning
- second and third delayed scans

Install:
1. Open chrome://extensions
2. Turn on Developer mode
3. Click Load unpacked
4. Select this folder
